FAQ

Every question
answered.

What exactly do I get when I buy? +
You get one or two HTML files (depending on what you purchase) delivered instantly via Gumroad. Open them in Chrome, Firefox, or Edge. No installation. No cloud account. No software to update. Everything runs entirely in your browser, privately, with no data leaving your device.
Do I need to be technical to use these? +
Not at all. If you can open a file in a browser, you can use these tools. The setup literally takes under 2 minutes. No coding, no configuration, no API keys required (unless you want to use the AI content features in SoloForge, which just needs a free Anthropic key).
What happens to my data? Is it private? +
100% private. All your data is stored in your browser's localStorage — on your own device. Nothing is sent to any server. No cloud storage. No third party ever sees your business data. You can also export your data as JSON at any time for safekeeping.
Can I use it on multiple computers? +
Yes. Download the file on each device. Data is stored locally on each machine and doesn't sync automatically. Use the export/import feature to move data between devices if needed.
What's the difference between FounderOS and SoloForge? +
FounderOS is the strategic layer — KPIs, OKRs, risk management, investor reporting. It's for running the company. SoloForge is the delivery layer — clients, invoices, tasks, AI content. It's for running the client work. Most founders benefit from both. The bundle gives you the complete system at a $17 discount.
Is there a refund policy? +
Yes. If the product doesn't work as described, contact within 7 days of purchase for a full refund. No questions asked if you have a genuine technical issue.
Does it work offline? +
Yes. After the first time you open the file, both tools work entirely offline. The only exception is the AI content repurposer in SoloForge, which requires an internet connection to call the Claude API.
